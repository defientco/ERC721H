### Getting Started

- install [forge](https://mirror.xyz/crisgarner.eth/BhQzl33tthkJJ3Oh2ehAD_2FXGGlMupKlrUUcDk0ALA)

### Features

- ERC721AH: supports ERC721A
- ERC721CH: supports ERC721C
- ERC721ACH: supports both ERC721A & ERC721C
