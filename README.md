# ERC721H

- extends [ERC721C](https://github.com/limitbreakinc/creator-token-contracts)

### Getting Started

- install [forge](https://mirror.xyz/crisgarner.eth/BhQzl33tthkJJ3Oh2ehAD_2FXGGlMupKlrUUcDk0ALA)
